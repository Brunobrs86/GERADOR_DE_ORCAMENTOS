# Gerador de Orçamentos para Autônomos

## 📝 Descrição do Problema Real
Profissionais autônomos e freelancers perdem tempo e profissionalismo ao enviar orçamentos desorganizados. Existe a necessidade de organizar rotinas e facilitar tarefas de precificação.

## 💡 Proposta da Solução
Aplicação CLI em Python que permite cadastrar a descrição de um projeto, valor da hora técnica e tempo estimado para gerar um resumo de orçamento padronizado.

## 🎯 Público-Alvo
Trabalhadores autônomos e microempreendedores.

## ✨ Funcionalidades Principais
- Interface de Linha de Comando (CLI).
- Cálculo automático de valores.
- Validação de entradas (impede valores negativos).
- Exibição de resumo formatado.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python 3.x.
- **Linting:** Ruff (Análise estática).
- **Testes:** Pytest (Testes automatizados).
- **CI:** GitHub Actions.

## 🚀 Como Instalar e Executar
1. Tenha o Python 3.10+ instalado.
2. Clone o repositório: `git clone https://github.com/Brunobrs86/GERADOR_DE_ORCAMENTOS.git`
3. Instale as dependências: `pip install -r requirements.txt`.
4. Execute: `python src/main.py`.

## 🧪 Como Rodar os Testes
Execute o comando: `pytest`.

## 🔍 Como Rodar o Lint (Análise Estática)
Execute o comando: `ruff check .`.
